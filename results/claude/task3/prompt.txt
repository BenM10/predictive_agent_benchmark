Task 3 — Baseline Model Training

You are a data scientist working in Python within a project repository.
The dataset is located at:
data/processed/Claude_clean.csv

The goal is to build a baseline machine learning model to predict passenger satisfaction.
Write Python code that:
1. Loads the dataset
2. Remove the 'id' column
3. Prepares the data for modelling (handle categorical variables if needed)
4. Splits the dataset into training and test sets (Random state = 42, train = 0.8, test = 0.2)
5. Trains a baseline classification model (Recommended logistic regression and optionally a
   RandomForestClassifier model and HistGradientBoosting model)
6. Evaluates the model using appropriate metrics (ROC AUC, F1, accuracy)
7. Include figures visualising these metrics

Use common Python machine learning libraries (e.g. pandas, scikit-learn).
Ensure evaluation is performed on a held-out test set.
The goal is to produce a simple, valid baseline model that could serve as a starting point
for further model development.

Execution Environment
The repository contains the following structure:
- data/processed/        -> contains the dataset
- notebooks/task3/       -> location where the experiment notebook should be saved
- results/Claude/task3/  -> location where experiment outputs should be logged

Ensure that all file paths are relative to the project root.
