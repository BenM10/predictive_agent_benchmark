# Task 4 — Improving Model Performance

## Objective

Evaluate whether the agent can improve the predictive performance of an existing baseline model by applying more advanced modelling techniques.

## Dataset

Airline Passenger Satisfaction dataset:

data/processed/airline_passenger_satisfaction_full.csv

The target variable is passenger satisfaction.

## Expected Steps

A successful solution should typically:

- Load the dataset
- Prepare features for modelling
- Train an improved classification model
- Apply techniques to improve performance (e.g. model choice, feature engineering, tuning)
- Evaluate model performance using appropriate classification metrics
- Compare the improved model to the baseline approach

## Expected Outputs

- Trained machine learning model
- Evaluation metrics (e.g. accuracy, F1-score, ROC-AUC)
- Optional visualisations such as confusion matrix or feature importance
- Evidence that model performance has improved compared to the baseline

## Success Criteria

A successful submission should:

- Produce runnable Python code
- Train a valid classification model
- Demonstrate improved predictive performance
- Follow sound machine learning practices
- Provide interpretable outputs and evaluation metrics
